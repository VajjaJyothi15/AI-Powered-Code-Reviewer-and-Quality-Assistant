# AI Powered Code Reviewer and Quality Assistant

## Project Overview
The **AI Powered Code Reviewer and Quality Assistant** is a Python project developed to analyze Python code and improve documentation quality.  

The system evaluates code quality, validates docstrings, and generates documentation using standard formats.

The project is developed in **four milestones**, each implementing a different functionality of the code review system.

---

## Project Features

- Code metrics analysis
- Docstring validation
- Automatic docstring generation
- Code review dashboard
- Support for standard docstring formats:
  - Google Style
  - NumPy Style
  - reStructuredText (reST)

---
## 🛠 Technologies Used

- **Python**
- **Streamlit**
- **pydocstyle**
- **pytest**
- **JSON**

---

## Project Milestones

### Milestone 1 – Code Metrics
This module analyzes Python files and calculates basic **code metrics**, such as:

- Function detection
- Line count
- Code structure analysis

File:
```
milestone1.py
```

---

### Milestone 2 – Docstring Validation
This milestone validates docstrings using **pydocstyle** and checks whether documentation follows proper formatting rules.

Features:
- Detect missing docstrings
- Validate docstring structure
- Report documentation issues

File:
```
milestone2.py
```

---

### Milestone 3 – Docstring Generation
This module automatically **generates docstrings** for functions using standard formats.

Supported formats:
- Google style
- NumPy style
- reStructuredText style

File:
```
milestone3.py
```

---

### Milestone 4 – Dashboard
This milestone provides a **dashboard interface** for viewing code review results and analysis.

Features:
- Code review results visualization
- Function analysis
- Documentation insights

File:
```
milestone4.py
```
